# 마켓컬리 클론코딩


## 1. 개요
- 목적 : 클론코딩을 통한 Node.js API 서버 구축
- 주요 개발 기능 : 회원 가입/로그인, 상품 검색, 주문하기
- 기간 : 2022.03 ~ 2022.04 (2주)
- 인원 : 서버(본인), iOS(익명 2명)
